# `analysis/` — read a pool, write a figure

Each script in this folder loads a pre-computed pool
(`simulation_data/fractal_dim_pool.npz` plus `.meta.json`) generated by
one of the `coding/simulators/` scripts, then emits a CSV summary +
a talk-grade PNG.

| script | what it does | output |
| --- | --- | --- |
| `masterpiece_plot.py` | RMSE of $\hat d_f$ vs theoretical budget; four $m_0$-schedules overlaid, one panel per $n$. | `images/fig_masterpiece.png`, `simulation_data/masterpiece.csv` |
| `mae_vs_time_plot.py` | $\langle\|\hat d_f - d_f\|\rangle$ vs wall-time seconds (calibrated from pool meta). | `images/fig_mae_vs_time.png`, `simulation_data/mae_vs_time.csv` |
| `regime_sweep.py` | Per-regime bias / std curves vs $m$, one column per $n$. | `images/fig_regime_sweep.png`, `simulation_data/regime_sweep.csv` |
| `error_vs_time.py` | $\|\text{bias}\|$ vs simulation time per replica, on log-log, with OLS slope per regime. | `images/fig_error_vs_time.png` |
| `scale_of_scales.py` | Cross-scale ratios $V(\rho^{k+1}) / V(\rho^k)$ vs $k$; check for finite-size bias at the largest scales. | `images/fig_scale_of_scales.png` |
| `verify_error_vs_time.py` | Numerical sanity-check: recomputes the §6 / §8 numbers two different ways and prints discrepancies. No plot. | stdout only |
| `plot_fractal_dim.py` | Universality plot — hex / square / triangular lattices on the same axes. Uses `fractal_dim_data.csv` from the older per-lattice simulator. | `images/fig_fractal_dim.png` |

## Path conventions

Each script starts with:

```python
import sys
sys.path.insert(0, str(Path(__file__).resolve().parent.parent))  # so `from lib.*` works
from lib.stats   import ...
from lib.plotting import ...

HERE = Path(__file__).resolve().parent           # coding/analysis
ROOT = HERE.parent.parent                         # repo root
DATA_DIR = ROOT / "simulation_data"
IMG_DIR  = ROOT / "images"
```

So scripts can be invoked from any CWD; everything resolves off
`__file__`. Output goes to `<repo>/images/` and `<repo>/simulation_data/`.

## Usage

```bash
# regenerate the four §7-§8 figures from the canonical pool
python3 coding/analysis/masterpiece_plot.py
python3 coding/analysis/mae_vs_time_plot.py
python3 coding/analysis/regime_sweep.py
python3 coding/analysis/error_vs_time.py
```

If the pool NPZ is missing the scripts print a friendly
`missing simulation_data/fractal_dim_pool.npz; run fractal_dim_pool_sim_par.py first`
and exit. Generate the pool with
[`../scripts/run_overnight.sh`](../scripts/run_overnight.sh).
